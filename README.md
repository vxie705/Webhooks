# Webhooks
Webhook Hub es una plataforma de infraestructura core diseñada bajo una Arquitectura Orientada a Eventos (EDA) para la ingesta, procesamiento y entrega de webhooks a escala masiva, soportando cargas proyectadas de 1,000 a 10,000+ RPS con una latencia p99 &lt; 100 ms y disponibilidad objetivo del 99.95%.
